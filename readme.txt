== Description ==

This widget will display your Google Latitude location badge on your blog which display's your current location. The map type, zoom level, and dimensions of the badge are configurable.

== Installation ==

1. Sign up for Google Latitude at [http://www.google.com/latitude](http://www.google.com/latitude "Google Latitude").
2. Go to Appearance -> Widgets -> Drag Google Latitude Badge Widget to your widget area.
3. Go to [http://www.google.com/latitude/apps/badge](http://www.google.com/latitude/apps/badge "Goolge Latitude User Id") and copy your user id from the code box under the Google Public Location Badge. Please see screenshot-1.jpg for an example.
4. Adjust the other widget settings to your liking and you're done.
